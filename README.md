# Project Acceloka

Untuk menjalankan codingannya :
1. di acceloka-frontend buka terminal menggunakan ctrl + ` lalu ketik npm install, pastikan masuk ke dalam path acceloka-frontend, jika belum bisa tulis cd acceloka-frontend
2. Untuk menjalankan frontend menggunakan npm run dev
3. Lalu untuk localhostnya, disini saya menyesuaikan dengan locaalhost saya, jika misalnya ketika menjalankan backend localhostnya berbeda maka nanti di code acceloka-frontend perlu diganti semua port localhostnya.
4. Jika nanti di dalam aplikasinya ada loading, artinya sistem masih mengambil data dari backend jadi bisa ditunggu untuk beberapa detik.
